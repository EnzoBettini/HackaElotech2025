DiagnoMap

Agente de IA - Saúde

Documentação para o backend de análise de dados de saúde.

Como Rodar

Siga os passos abaixo para configurar e iniciar o servidor do Agente de IA:

  1. Instale as dependências: 
    npm install
  2. Inicie o servidor: 
    npm start

Após iniciar, o servidor estará disponível em: http://localhost:3001

Endpoint de Análise

O agente expõe um endpoint principal para processar os dados de saúde:

POST /analise

Corpo da Requisição (Body):

Envie um JSON no seguinte formato:

{
    "dados": [
        {
            "endereco": "Rua X, Bairro Y",
            "diagnostico": "Dengue",
            "sintomas": "febre, dor de cabeça"
        },
        // ... mais objetos de dados aqui
    ]
}

Resposta Esperada:

O servidor responderá com um JSON contendo o resultado da análise:

{
    "resultado": "...resposta do Gemini em formato markdown..."
}

A chave "resultado" conterá a análise textual, que pode incluir formatação Markdown para melhor apresentação.
